collectd-smartmon
================
This is a python exec plugin for collecting SMART disk metrics.

Requirements
------------
*collectd*  
smartmontools installed (and smartctl binary)
sudo entry for binary (ie. for sys account):      sys   ALL = (root) NOPASSWD: /usr/sbin/smartctl
